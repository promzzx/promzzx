## About Me

Backend Engineer with **5+ years of commercial experience** designing and operating high-load systems in production. I write Go, think in distributed systems, and treat observability as a first-class concern — not an afterthought.

I've shipped services processing **300–1500 RPS**, integrated **20+ external systems** through async pipelines, and dealt with the real problems that come after deployment: latency spikes, data consistency under failure, and cascading failures in distributed environments.
