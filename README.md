## About Me

Skilled Backend / Fullstack Engineer with **5+ years of commercial experience** building scalable, high-load, fault-tolerant distributed systems. Experienced in integrating **20+ external services**, maintaining SLA under sustained load, and working with large datasets where availability is non-negotiable.

Strong background in **Go**, system design, and data-intensive applications — asynchronous processing, caching strategies, and microservice architecture included. Extended into **frontend** (React, React Native) and **Web3** (Solidity, EVM, Solana, ZKP). Familiar with the full DevOps cycle: containerization, CI/CD, cloud infrastructure, and production observability.

Focused on performance, reliability, and solving problems that actually matter at scale.
